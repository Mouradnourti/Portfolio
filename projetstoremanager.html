StoreManager

Application desktop développée en Java avec une interface JFrame/Swing. StoreManager permet de gérer les produits, le stock, les ventes, les commandes, les factures, les employés, les fournisseurs et les clients depuis un tableau de bord simple.

Présentation du Projet

Une application de gestion commerciale créée avec Java Swing pour centraliser les opérations d'un magasin : stock, ventes, commandes, factures, employés, fournisseurs et clients.

Tableau de bord StoreManager

L'écran principal permet d'accéder rapidement aux fonctionnalités principales : stock, ventes, commandes, factures, employés, fournisseurs et clients. L'interface est conçue avec JFrame pour offrir une navigation claire entre les fenêtres.
